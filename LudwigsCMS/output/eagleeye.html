The Eagle Eye Project.

• Used the Unity Game Engine to create a web app that gave our company’s production operators finer control over processing live events. • Created native plugins for Unity, using C++, to integrate the Zoom meeting SDK, BEFORE Zoom themselves offered the video SDK as a plugin for Unity. • Utilized Unity’s WebRTC plugin to create a sophisticated set of live audio and video tracks that let operators: ◦ Preview Zoom audio output in real time and separated from any other audio (i.e. background music). ◦ See Unity’s video output in real time. ◦ See a live dashboard (rendered in Unity) of all Zoom meeting participants that multiple operators could micro manage at once.
